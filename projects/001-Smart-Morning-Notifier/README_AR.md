# 001 - المنبه الذكي الصباحي

## الوصف
نظام آلي يرسل مهامك اليومية إلى تيليجرام كل صباح الساعة 8:00. المهام مرتبة حسب أيام الأسبوع مع اقتباسات تحفيزية عشوائية.

## الأدوات المستخدمة
- N8N
- Telegram Bot
- JavaScript / Python

## طريقة الاستخدام
1. حمّل ملف smart-morning-notifier.json.
2. استورده في N8N.
3. أضف توكن بوت تيليجرام و Chat ID.
4. خصص المهام والاقتباسات من عقدة Code/Python.
5. فعّل الـ Workflow.

## الجدول
- من الاثنين إلى الأحد الساعة 8:00 صباحاً.

## التخصيص
عدّل مصفوفة tasks في عقدة Code لتناسب روتينك الأسبوعي. غيّر الاقتباسات كما تحب.
